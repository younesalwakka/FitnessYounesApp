# FitnessYounesApp

Bu proje Web Programlama dersi icin yapilmaktadir.  
Projenin amaci bir spor salonu icin yonetim sistemi hazirlamaktir.  
Salon, antrenor, hizmetler, uyelik ve randevu islemleri bu projede yer almaktadir.

Ayrica projede yapay zeka kismi da olacak.  
Kullanicilar istedikleri zaman vucut bilgilerini girip ya da fotograf yukleyip egzersiz veya diyet oneri alabilecektir.

Projeyi ASP.NET Core MVC kullanarak gelistiriyorum.  
Veritabani icin Entity Framework Core ve SQL Server kullaniyorum.  
Arayuz tarafinda ise Bootstrap'ten yararlaniyorum.

Projeyi gelistirirken GitHub'i duzenli olarak kullanmaya calisacagim.  
Ders kapsaminda hem kodlari hem de gelistirme surecimi buradan paylasacagim.
